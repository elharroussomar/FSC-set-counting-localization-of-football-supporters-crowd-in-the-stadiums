# FSC-set-counting-localization-of-football-supporters-crowd-in-the-stadiums
The FSC-Set (Football Supporters Crowd Dataset) is a large-scale collection of images specifically designed for crowd counting and individual localization in highly dense stadium environments.

This dataset addresses the critical scarcity of annotated data for crowd scene analysis, particularly in sports venues, providing essential resources for research in areas like public safety, crowd management, and video surveillance systems.
